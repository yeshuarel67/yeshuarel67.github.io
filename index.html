APCSP Unit 1 – The Internet & Networking

Devices communicate using protocols like TCP/IP. Every device has an IP address, and the Domain Name System (DNS) translates names into those addresses. Data is divided into packets, which improves fault tolerance and redundancy. Bandwidth determines the speed at which packets travel across the network.

APCSP Unit 2 – Data & Algorithms

Data is represented in binary and simplified with abstraction. Algorithms provide step-by-step instructions for solving problems and can be tested with simulation. Files often include metadata to describe their contents. To reduce file size, we use compression—either lossless (no information lost) or lossy (some details removed for efficiency).

Virtual Reality on the Web

Virtual Reality (VR) is fully immersive and comepletely computer generated. Augmented Reality (AR) is like VR, but adds digital features to the real world. These standards let developers create immersive 3D environments that run in a browser without extra software.

Web-based VR/AR relies on the same Internet fundamentals: bandwidth for smooth graphics streaming, protocols for secure communication, and compression to deliver large 3D models efficiently.
